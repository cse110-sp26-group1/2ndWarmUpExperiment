User: Linda (budget-conscious player)

User Story:
Linda enjoys playing slot games, but she does not want to lose track of how much she is spending. She wants the slot machine to clearly show her current balance, total amount bet, and total winnings during her session so she can make informed decisions while playing. This helps her feel more in control and makes the game easier to trust and use.
(Persistent UI stats panel, state management for session totals, responsive layout, real-time value updates)

- The interface always displays the player’s current balance.
- The interface shows the current bet amount before each spin.
- The interface tracks and displays total winnings for the session.
- After each spin, the displayed values update correctly and immediately.
- The stats remain visible while the reels are spinning.
- The layout is easy to read and does not hide important information behind extra menus.
